## Installation

This [Quarto](quarto.org) template can be installed/used using the following command:

- Install
  ```bash
  quarto add mcanouil/quarto-revealjs-storybook
  ```
- Use
  ```bash
  quarto use template mcanouil/quarto-revealjs-storybook
  ```
